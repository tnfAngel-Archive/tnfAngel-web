Coche RC Arduino Bluetooth

Coche robot con Arduino como cerebro controlado por Bluetooth.


Material necesario


Componentes Electrónicos:


  • Arduino UNO
  • (x2) Motor TT 130
  • Sensor ultrasónico HC-SR04
  • Puente en H L298N
  • Servomotor SG90
  • Módulo de bluetooth XM-15B SPP
  • Cables o Jumpers
  • (x3) Portapilas x4 AAA
  • (x12) Pilas AAA pequeñas
  • Interruptor basculante 2 pines
  • Clema

Elementos comprados:


  • (x4) Ruedas de coche para Motor TT 130
  • (x6) Tornillos 40mm x 4mm
  • (x6) Arandelas 4mm
  • (x6) Tuercas 4mm
  • (x11) Tornillos 40mm x 3mm
  • (x11) Arandelas x 3mm
  • (x11) Tuercas x 3mm
  • Brazo doble de servomotor

Piezas impresas en 3D:


  • Chasis parte de abajo (Chassis_DOWN)
  • Chasis parte de arriba (Chassis_UP)
  • (x2) Ejes de transmisión (Driveshaft_front)
  • Parachoques frontal (Frontal)
  • (x2) Buje delantero (Hub_front)
  • (x2) Pin de pivote (Pivot_hub)
  • (x4) Anillo Seeger (Seeger)
  • (x2) Espaciador (Spacer)
  • (x2) Barra de dirección (Steering Bar)
  • (x2) Arandelas (Washer)

Herramientas:


  • Impresora 3D
  • Destornillador

Software:


  • Arduino IDE
  • Mozilla Firefox
  • Ultimaker Cura
  • Bluetooth Electronics

Programación del Arduino


Librerías Utilizadas:


SoftwareSerial:

Librería utilizada para la comunicación entre el modulo de bluetooth XM-15B SPP y Arduino.


L298NX2:

Librería para usar el puente en H L298N en los dos motores fácilmente.


Servo:

Librería utilizada para el control del servomotor que usa la dirección.


EzBuzzer:

Librería para emitir sonidos con el buzzer pasivo sin bloquear los núcleos del Arduino y que se pueda ejecutar otras lineas de código en paralelo.


Pitches:

Librería con todos los tonos para poder reproducir sonidos con notas.



Código fuente programado del coche y librería pitches.
